Context

Регистровый контекст процесса
Содержимое всех регистров процессора, связанных с данным процессом. Данные аппаратуры.
Системный контекст прцесса
Часть PCB без регистрового контекста процессора. Включает все, что выдала процессу OS. Данные OS.
Пользовательский контекст процесса
Код и данные, находящиеся в адресном пространстве процесса (виртуальной памяти). Данные пользовательские.
Контекст
Содержимое регистрового, системного, пользовательского контекстов.

При выполнении процесса меняются значения регистров. Когда процесс уходит с процессора - контекст застывает. Он определяет, что должно быть в регистрах, когда процесс будет запущен на компьютере в следующий раз.

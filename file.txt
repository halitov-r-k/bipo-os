<h1> Файлы <h1>
Файл (F) - часть пространства на носителе информации именованная.
F область структурированная памяти внешней.
F состоит из набора записей - структуры с именованными полями. 
F имеет свою структуру записи.
F можно определять:
  последовательным - последовательность записей, когда 
   писать можно только в хвост памяти,
   читать можно только в начала.
  прямого доступа - позволяют обращаться к записи по ее номеру,
  индексно-последовательные - поле указывается ключевым, система позволяет обращаться к файлу, указывая значение поля ключевого, записи нужной.

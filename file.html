File (F)
часть пространств на носителе именованная;
Область структурированная памяти внешней;

F состоит из набора записей - структуры с именованными полями. F имеет структуру свою записи.

F можно определить:

  • последовательным (последовательность записей) когда писать можно в хвост F, читать только с начала F;
  • прямого доступа (обращение к записи по номеру (индексу));
  • индексно-последовательные (поле указывается ключевым, система позволяет обращаться к F, указывая значение поля ключевого записи нужной.

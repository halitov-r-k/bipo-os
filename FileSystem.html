File System

File System (FS) система файлов - часть ядра OS (операционной системы).

FS задачи: управление файлами, скрытие особенностей ввода|вывда, создание модели абстрактной файлов (независимость от устройств).

FS вызовы системные:

  • create ,
  • delete,
  • open,
  • close,
  • read,
  • write.

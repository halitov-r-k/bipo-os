OS

Структура вычислительной системы классы компонентов:

hardware
аппаратное обеспечение (процессор, память осноовная, устройство взаимодействия с пользователем, терминал, дисковые устройства, объединенные шиной, позволяющей процессору выполнять обмены с этими устройствами);
software
программное обеспечение прикладное и системное

Операционная система (OS)
системное программное обеспечение.

Точки зрения на OS:

  • виртуальная машина;
  • менеджер ресурсов;
  • защитник пользователей, программ;
  • ядро(kernel), функционирующее постоянно.

Функции основные OS классических:

  • планирование заданий, использования процессора;
  • обеспечение программ средствами коммуникации, синхронизации;
  • управление памятью;
  • управление системой файловой;
  • управление вводом-выводом;
  • обеспечение безопасности.
